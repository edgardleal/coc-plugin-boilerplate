# coc-plugin

my custom coc plugin

## Install

`:CocInstall coc-my-plugin`

### With Plug

`Plug 'username/coc-my-plugin', { 'do': 'make' }`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-my-plugin-keymap)`
